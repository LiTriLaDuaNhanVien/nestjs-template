## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation
### Auto Generate Swagger
### Hot reload
### ValidationPipe
```bash
$ pnpm install
```



## Running the app

```bash
# development
$ pnpm run start

# dev run
$ pnpm run dev

# watch mode`
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```

## Test

```bash
# unit tests
$ pnpm run test

# e2e tests
$ pnpm run test:e2e

# test coverage
$ pnpm run test:cov
```
